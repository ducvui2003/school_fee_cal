# Install Invoke

```cmd
    pip install invoke
```

## Show list task

```cmd
    invoke --list
```

# Setup Env

## 1. Create venv

```cmd
    invoke env
```

## 2. Install deps in `requirements.txt`

```cmd
    invoke install
```

# Run app

```cmd
    invoke main
```
